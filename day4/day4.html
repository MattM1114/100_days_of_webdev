Hosting

For a website to be accessible online it needs to be hosted. Hosting refers to storing files, code, and other content on a server that is connected to the internet. Some common hosting options include:

  • Github Pages - Free static website hosting through Github repositories
  • Netlify - Free static website hosting with continuous deployment from Git repos
  • Heroku - Free hosting for simple dynamic web apps, scales with paid tiers

When hosting a website, the files will be placed on the server and accessed via a domain name or IP address. For example, a site hosted on Github Pages may have a URL like myusername.github.io. The server handles requests for pages and assets and serves them to users visiting the site.
